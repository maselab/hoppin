Detailed Description

今回作成していただくプレイヤーの抽象クラスです． ゲーム側はGetMove()を使用してプレイヤーの動作を決定しています． 使用できる関数は参照値を返すため，取得したデータをそのまま変更すると，関数から取得できる値も変更されてしまいます．
